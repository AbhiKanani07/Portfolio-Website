# Abhi Kanani — Portfolio

A personal site showcasing data products, analytics case studies, and applied modeling projects. Built with HTML/CSS/JavaScript and a light/dark theme toggle.

## Live Sections
- Home: About, rotating updates, featured projects.
- Projects: Longer write-ups with links to detail pages.
- Sleep Pattern & Health Insights Dashboard: End-to-end Fitbit/Apple Health pipeline for sleep analytics.
- Water Quality in the Netherlands: Spatial/seasonal analysis with anomaly checks and decision-ready visuals.
- YouTube Recommendation Bias & Viewing Model: Graph-based “baby YouTube algorithm” using personal watch history.

## Project Links
- Sleep Dashboard: https://github.com/AbhiKanani07/Sleep-Dashboard
- Water Quality (live): https://abhikanani07.github.io/Dutch-Water-Model/
- YouTube Behavior Model: https://github.com/AbhiKanani07/YT-Behavior-Model

## Tech Stack
- HTML, CSS (`pros.css`)
- JavaScript (`app.js`) for theme persistence, rotating updates, and dynamic project cards
- No build step; pure static assets

## Run Locally
```bash
# clone
git clone https://github.com/AbhiKanani07/Portfolio-Website.git
cd Portfolio-Website

# open in browser
# on macOS
open index.html
# on Windows
start index.html
# on Linux
xdg-open index.html
Structure
index.html — About, featured projects, contact
projects.html — Project summaries
sleep-dashboard.html — Sleep case study
water-quality.html — Water quality case study
youtube-rec.html — YouTube recommendation bias case study
pros.css — Styling, gradients, light/dark theme
app.js — Theme toggle, rotating posts, project list injection
Notes
Theme preference is stored in localStorage.
Links to project pages open in the same tab; external resources open in a new tab.
Light mode uses orange accents only for internal page links to keep readability high.
